# Product Images

Add your product images to this folder.

## Required Images:

1. **active-contact-mic.jpg** - Active Magnetic Contact Microphone
2. **active-exciter.jpg** - Active Exciter System
3. **passive-contact-mic.jpg** - Passive Magnetic Contact Microphone
4. **soundsniffer.jpg** - SoundSniffer Kit
5. **placeholder.jpg** - Generic placeholder (for missing images)

## Image Guidelines:

- **Format:** JPG or PNG
- **Recommended size:** 1200x800px or similar aspect ratio
- **Quality:** High resolution, clear product shots
- **Background:** Dark or neutral backgrounds work best with the site theme
- **Naming:** Use lowercase with hyphens (as shown above)

## Tips:

- Consider showing products in use or from multiple angles
- Maintain consistent lighting across all product photos
- Add detail shots for close-ups of connectors, build quality, etc.
- Consider adding lifestyle shots showing the products in action

The site will automatically use placeholder.jpg if a product image is missing.
