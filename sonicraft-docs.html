Getting Started

SoniCraft is a progressive web app (PWA) that runs entirely in your browser. No downloads, no sign-ups. It works on phones, tablets, and desktops.

Install as an app

For the best experience, install SoniCraft to your home screen. Open the hamburger menu and tap Install app, or use your browser's "Add to Home Screen" option. This enables offline use and removes the browser chrome.

Interface overview

The app has four modes, selectable from the tabs at the top of the screen:

  • REC -- Record audio and edit waveforms
  • SAMPLE -- Play slots as a 16-pad sampler/drum machine
  • SEQ -- Program and play step sequences
  • GEN -- Use video or camera analysis to drive the sampler

Below the mode tabs is a 4x4 slot grid. Each slot holds one audio sample (or a drum kit). The toolbar sits below the waveform and changes depending on the active mode.

Slots & Banks

SoniCraft has 16 audio slots arranged in a 4x4 grid. Slots are grouped into 4 banks of 4, each colour-coded (red, green, blue, yellow).

Slot actions

  • Tap an empty slot to select it, tap again to start recording
  • Tap a filled slot to select it, tap again to play/stop
  • Long-press (or right-click) a slot for the context menu: Rename, Duplicate, Save to device, Resample, Make Drum Kit, Clear
  • Double-tap a kit slot to enter drum kit mode

Each slot shows a mini waveform thumbnail, its name, and its number (01-16).

REC Mode

Record audio from your device's microphone or an external audio input. The toolbar provides transport controls and editing tools.

Recording

  • Select an empty slot, then tap REC to start recording
  • Tap the slot again (or REC) to stop
  • A live waveform and timer display while recording
  • After recording, a rename dialog appears

Noise gate

Tap GATE to enable the noise gate. A threshold slider appears (-60 to 0 dB). Audio below the threshold is silenced, useful for cutting background noise between sounds. The REC button dims when the gate is closed.

Input selection

Open the hamburger menu and tap Audio input to choose between available microphones or USB audio interfaces.

Playback

  • PLAY -- Play the current slot from the cursor or start of selection
  • LOOP -- Toggle looped playback of the selection or full sample

Waveform interaction

  • Drag on the waveform to make a selection
  • Tap to position the playback cursor
  • Long-press (right-click) the waveform for a context menu with Cut, Copy, Paste, Silence, Fade In, Fade Out, and more
  • Use the zoom buttons to zoom in/out, or FIT to show the full sample

Loading files

Tap LOAD in the toolbar to import a WAV or other audio file from your device into the selected slot.

Editing & Processing

Toolbar tools

  • TRIM -- Trim the audio to the current selection
  • NORM -- Normalise the audio to peak amplitude
  • REV -- Reverse the audio (or just the selection)

Edit group

Tap EDIT in the toolbar to expand the clipboard tools:

  • CUT -- Cut the selection to the clipboard
  • COPY -- Copy the selection to the clipboard
  • PASTE -- Paste the clipboard at the cursor position

Process group

Tap PROCESS to expand the time/spectral processing tools. Each opens a dialog with parameter sliders and a preview button:

PAUL

Paulstretch. Extreme time-stretch for ambient textures. Stretch: 2-50x, Window: 0.1-1.0s.

GRAIN

Granular Freeze. Freeze a moment and scatter it. Position, grain size, density, duration.

SPEC

Spectral Freeze. Hold the frequency spectrum at a point. Position, duration, smoothing.

STRETCH

Time Stretch. Speed up or slow down without changing pitch. Rate: 0.25x-4x.

PITCH

Pitch Shift. Shift pitch without changing speed. Range: -12 to +12 semitones.

FOLD

Wavefolding. Adds harmonic distortion by folding the waveform. Threshold and gain controls.

RING

Ring Modulation. Multiply by a sine wave for metallic/bell tones. Frequency and mix.

GLITCH

Stutter / Glitch. Chop and repeat slices. Slice size, repeat count, scatter amount.

Cross-slot processing

Available from the waveform context menu or SAMPLE mode FX panel:

  • Cross -- Convolve, ring modulate, or vocode one slot with another
  • Layer -- Mix multiple slots together with per-slot volume and pan, then bounce to a new slot
  • Bounce -- Flatten the current audio (including effects) to a new slot

Undo / Redo

Up to 5 levels of undo. Use the arrow buttons in the toolbar. The counter shows how many undo steps are available.

SAMPLE Mode

Turns the 16 slots into a playable sampler/drum machine. Each pad has its own playback mode, envelope, filter, LFO, and pitch controls.

Playing pads

Tap pads on screen or use the keyboard:

RowKeysPads
11 2 3 41-4
2Q W E R5-8
3A S D F9-12
4Z X C V13-16

Pad modes

  • ONE -- One-shot. Plays the sample once from start to end. Retrigger restarts.
  • LOOP -- Loops continuously between the loop markers.
  • GATE -- Hold to play, release to stop (with release envelope).
  • MORPH -- Blend between two samples using spectral or ring modulation algorithms.

Toggle REV to play any pad in reverse.

AMP envelope (ADSR)

Each pad has independent Attack, Decay, Sustain, and Release controls. Adjust these to shape the volume curve of each sample -- from punchy drum hits (short attack/decay) to swelling pads (long attack).

Pitch

Shift each pad's pitch from -24 to +24 semitones. An optional pitch envelope adds pitch sweep over time.

Filter

Per-pad biquad filter with lowpass, highpass, and bandpass types. Control frequency (20 Hz - 20 kHz) and resonance (Q).

LFO

Per-pad low-frequency oscillator that modulates a target parameter:

  • Filter -- Wobble the filter cutoff
  • Volume -- Tremolo effect
  • Pitch -- Vibrato effect
  • Position -- Scrub the playback position for granular-like textures

Choose from sine, triangle, square, or sawtooth shapes. Adjust rate (1-200 Hz) and depth (0-100%).

Morph

Select a target slot and a morph algorithm (Ring Mod, AM, Spectral Morph, Phase Swap, Spectral Gate). The amount slider blends between the original and the morphed result.

Chromatic keyboard

Tap KEYS to overlay a chromatic piano keyboard on the waveform. Click keys to trigger the selected pad at different pitches. Adjust the base octave to change the range.

Region and loop markers

In SAMPLE mode, dragging a selection on the waveform sets the playback region (for one-shot/gate) or the loop points (for loop mode). Changes apply in real-time to playing voices.

SEQ Mode

A step sequencer for programming rhythmic patterns with your samples.

Transport

  • PLAY -- Start/stop the sequence
  • REC -- Enable live recording: play pads or keys while the sequence runs and your input is captured to steps
  • UNDO -- Remove the last recording layer

BPM

Tap the BPM display to type a value (20-300). Use the +/- buttons for fine adjustment, or tap TAP repeatedly to set tempo by feel.

Pattern editing

  • The grid shows slots as rows and steps as columns
  • Tap a step to toggle a sample on/off
  • Long-press a step for per-step options: playback mode (one-shot/loop), direction (forward/reverse), and pitch offset
  • Multiple samples can play on the same step

Pattern length

Patterns can be 16, 32, 48, or 64 steps. Longer patterns scroll vertically.

Pattern tools

  • REV -- Play the pattern backwards
  • SPEED -- Playback speed: 0.5x, 1x, 2x, 4x
  • RAND -- Randomise the pattern
  • CLEAR -- Clear all steps

Stutter and Mutate

  • STUT -- Add stutter/glitch effects to the pattern. Amount slider controls intensity. Can be toggled per-slot.
  • MUTATE -- Evolve the pattern over time. Small random changes accumulate at each loop. Amount slider controls how much changes. Per-slot control available.

Pattern banks

16 pattern banks (A1-D4) let you store different patterns. Switch banks to change the pattern. Banks can be queued to switch at the next loop boundary for seamless transitions.

Per-slot controls

Each slot row has mute and solo buttons, plus per-slot stutter (ST) and mutate (MT) toggles.

Bounce sequence

Record the sequencer output to a slot. Set the number of loops to capture, then the result is saved as a new audio slot you can edit or process further.

GEN Mode

Use video files or your device camera as a controller for the sampler. Analyse regions of the video frame and map visual properties to sound parameters.

Video source

  • FILE -- Load a video file from your device
  • CAM -- Use a live camera feed (select from available cameras with the dropdown)

Video controls

  • PLAY / STOP -- Control video playback
  • LOOP -- Toggle loop mode
  • IN / OUT sliders -- Set the loop region within the video
  • Expand -- Toggle full-screen video view

Sensor regions

Draw rectangular regions on the video overlay. Each sensor analyses a visual metric and maps it to a sampler parameter on a selected pad.

Metrics:

  • Brightness -- Average luminance of the region
  • Hue -- Dominant colour tone
  • Saturation -- Colour intensity
  • Contrast -- Variation in pixel brightness
  • Motion -- Frame-to-frame movement detection

Targets:

  • Volume -- Control pad volume
  • Pitch -- Control pad pitch
  • Filter -- Control filter frequency
  • Trigger -- Trigger/gate the pad when threshold is crossed
  • V-POS -- Drive the video loop position itself
  • V-SIZE -- Drive the video loop size

Threshold

Each mapping has a threshold slider (0-0.99). For trigger targets, this sets the activation point. For continuous targets, it sets a floor -- values below the threshold are ignored and values above are rescaled.

Drum zones (kit mode)

When a kit slot is active, Gen mode can use Zones -- rectangular regions on the video that trigger specific kit pads when motion is detected. Draw zones on the overlay, assign each to a kit sub-slot, and set a motion threshold. Zones are saved per kit slot.

Camera recording

In CAM mode, tap the REC button to capture a video clip from the camera for analysis.

Drum Kits

Any slot can be converted into a drum kit containing 16 sub-samples.

Creating a kit

Long-press an empty slot and choose Make Drum Kit. The slot becomes a kit container. Double-tap it to enter kit mode.

Kit mode

Inside a kit, the 16-pad grid represents the kit's sub-samples. Record, import, or edit each pad individually. All sampler features (ADSR, filter, LFO, pitch) apply per kit pad.

  • Use PAD PLAY for single-tap triggering with velocity from touch pressure
  • Tap BACK to exit kit mode

Kits in the sequencer

When sequencing a kit slot, the grid shows a drum-style view with each kit pad as a row. Steps can reference specific kit sub-samples.

Removing a kit

Long-press the kit slot and choose Unmake Drum Kit. This deletes all sub-samples and converts the slot back to a regular empty slot.

Resample

Capture the master audio output into a slot. Useful for bouncing live performances, recording loop manipulations, or freezing effects.

How to resample

  1. Long-press an empty slot and choose Resample
  2. The slot turns amber and pulses -- recording has started
  3. Switch to any mode (SAMPLE, SEQ, GEN) and perform
  4. Tap the pulsing amber slot to stop -- the audio is saved and you're returned to REC mode

Resampling captures everything going through the master output, including live effects (reverb, delay, filter). The recording is clean -- it taps the audio graph directly with no mic noise.

MIDI

SoniCraft supports MIDI input and output for pads, sequencer, and clock sync.

Setup

Open the hamburger menu and tap MIDI. Select your input and output ports, and set the MIDI channel (1-16, or Omni to receive on all channels).

Note mapping

  • GM Drums -- Notes 36-51 map to pads 1-16 (standard General MIDI drum layout)
  • Launchpad -- Custom mapping for Novation Launchpad Mini (4x4 grid)

Clock sync

  • Off -- No clock sync
  • Send -- SoniCraft sends MIDI clock to external gear
  • Receive -- SoniCraft follows external MIDI clock for sequencer tempo

Projects & Export

All data is saved automatically in your browser's storage. Use project files to back up or transfer between devices.

Save project

Hamburger menu > Save project. Creates a .sonicraft file (ZIP) containing all slots, kit samples, sequencer patterns, sampler settings, and Gen configuration. On mobile, the share sheet opens so you can save to Files, cloud storage, or send directly.

Load project

Hamburger menu > Load project. Select a .sonicraft file to restore. This replaces all current data.

Export individual slots

  • SAVE button in toolbar -- Export the current slot as a WAV file
  • Long-press a slot > Save to device -- Same as above
  • Hamburger menu > Export all slots -- Batch export all non-empty slots as individual WAV files

Tips

  • Offline use: Install as a PWA for full offline support. All audio is stored locally in your browser.
  • Copy selections to slots: In REC mode, select a region on the waveform, then tap an empty slot to copy the selection there. Great for slicing samples.
  • Live effects: In SAMPLE mode, the FX panel provides real-time reverb, delay, filter, and more that apply to pad playback and sequencer output.
  • Resample your performance: Use Resample to capture a live jam into a slot, then edit it, process it, or slice it into a kit.
  • Generative drums: Load a video, set up motion-triggered zones mapped to a drum kit, and let the video play your drums.
  • Mutate patterns: In SEQ mode, enable MUTATE with a low amount for subtle evolving patterns, or high for chaos.
  • Paulstretch everything: Record a short sound, use PAUL with high stretch values, and you have an ambient drone.
  • Browser storage: Your data lives in the browser. Clearing site data will delete everything -- use Save Project to back up regularly.
