Connecting to the Patcher ​

The patcher communicates with your Smart Contact Mic over USB using Web Serial. No drivers or plugins are needed -- it runs entirely in your browser.

Connect ​

  1. Open the patcher at soniphorm.com/app
  2. Plug your board in via USB-C
  3. Click the Connect button in the toolbar (plug icon)
  4. A browser dialog will appear listing available serial ports -- select your ESP32-S3 device
  5. The connect button will change to show a connected state

Browser Support

Web Serial is supported in Chrome, Edge, and Opera on desktop. It is not available in Firefox, Safari, or any mobile browser. On mobile devices, you can use BLE (Bluetooth) to connect instead.

Verify the Connection ​

Once connected, open the Serial Console from the toolbar. Type STATUS and press Enter. You should see a response showing the firmware version, CPU usage, and current patch info.

Upload a Patch ​

With the connection active:

  1. Build a patch on the canvas (or load an example from the menu)
  2. Click the Upload button in the toolbar (arrow icon)
  3. The patch uploads as JSON over serial -- you'll hear the result immediately
  4. The serial console shows upload progress and confirmation

Disconnect ​

Click the connect button again to disconnect, or simply unplug the USB cable. The board continues running the last uploaded patch independently.

Troubleshooting ​

ProblemSolution
No ports listed in the dialogCheck the USB cable. Press RESET on the board. Try a different USB port.
Connection drops frequentlyUse a shorter USB cable. Avoid USB hubs.
Upload fails or times outOpen the serial console and check for error messages. Try disconnecting and reconnecting.
"Failed to open serial port"Another app (Arduino IDE, serial terminal) may be using the port. Close it and retry.
